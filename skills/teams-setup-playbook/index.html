01 — O QUE É UMA SKILL

Não é um chatbot.
É um especialista configurado
para o seu contexto.

Uma skill é um módulo que você instala no Cowork — o app desktop da Anthropic para automação de tarefas. Quando ativada, ela muda como o Claude se comporta: em vez de você precisar explicar o contexto toda vez, a skill já sabe o que perguntar, como estruturar a resposta e quais entregáveis gerar.

Sem skill, você abre o Claude, olha para a tela em branco e pensa: "o que eu digito?". Com a skill instalada, você digita uma frase — e o Claude faz as perguntas certas, conduz o processo e entrega material pronto.

É a diferença entre ter um estagiário que espera instrução e ter um consultor que já conhece o seu problema.

ANALOGIA DIRETA

"É como instalar um plugin no seu editor de código — só que o plugin ensina o Claude a trabalhar do jeito certo para a sua realidade."

Você instala uma vez. O protocolo funciona sempre que você precisar. Sem configuração manual, sem prompts longos para decorar.

02 — POR QUE ISSO IMPORTA

Sem padrão,
cada pessoa usa do seu jeito.

SEM SKILL

Cada um faz do seu jeito

  • Ninguém sabe exatamente como pedir o que precisa
  • Os resultados variam muito de pessoa para pessoa
  • Reuniões para explicar "como usar o Claude"
  • Tempo perdido tentando lembrar qual prompt funcionou
  • Adoção irregular — alguns usam todo dia, outros nunca
COM SKILL

Sistema. Padrão. Entregável.

  • Infraestrutura montada em 7 dias com guia passo a passo
  • System prompts prontos para cada Project do time
  • Templates que qualquer pessoa do time consegue usar
  • Kit de comunicação para o rollout — mensagens prontas
  • Onboarding documentado, replicável e sem fricção
03 — O QUE ESTA SKILL ENTREGA

Cinco entregáveis.
Todos customizados para o seu time.

01

Arquitetura de Projects

Estrutura de pastas e nomes para organizar o Claude Teams. Quais Projects criar, como nomear, quem tem acesso a quê.

02

System Prompts

Instruções globais prontas para colar em cada Project. Define o tom, o contexto e o comportamento do Claude para cada área do time.

03

Prompt Templates

Frases curtas e diretas que qualquer pessoa do time usa para gerar os entregáveis mais comuns — sem precisar saber escrever prompts.

04

Kit de Comunicação

Mensagens prontas para Slack, DM e email de rollout. Apresenta o sistema para o time sem causar resistência nem confusão.

05

Guia de Onboarding

Documento completo e pronto para enviar ao time na sexta-feira. Cobre o quê, por quê e como — do zero ao primeiro uso em menos de 10 minutos.

04 — COMO INSTALAR E USAR

Três passos.
O Claude faz o resto.

01

Baixe o arquivo

Clique em "Baixar a Skill" e salve o arquivo .skill em qualquer pasta do seu computador.

02

Instale no Cowork

Clique duas vezes no arquivo ou arraste para dentro do Cowork. A skill aparece instalada em segundos.

03

Acione e deixe trabalhar

Digite quero implantar o Claude no meu time — a skill assume o comando, faz as perguntas certas e gera tudo.

Requer Cowork instalado e plano Claude Teams (mínimo 5 seats). O Cowork está disponível em claude.ai/download.

Sete dias para o time trabalhar com IA de verdade.
